独特、生产级的前端界面，摒弃了通用的 AI 美学。

在编码之前引导设计思维：确立大胆的美学方向（粗犷主义、极繁主义、复古未来主义、奢华、趣味等），并明确目标、基调和差异化。

强调字体排版、色彩主题、动态效果、空间构图和纹理细节作为核心设计支柱；运用 CSS 变量、动画和滚动触发效果。

生成可运行的功能性代码（HTML/CSS/JS、React、Vue），使实现复杂度与美学理念相匹配：对于极繁主义设计，生成精细复杂的代码；对于极简主义设计，生成精简而精准的代码。

明确避免使用通用模式：过度使用的字体（Inter、Roboto）、陈词滥调的配色方案（紫色渐变）、可预测的布局以及千篇一律的组件设计。
