# Dream Log 梦境记录系统完整规划

## 一、核心需求分析与用户心理洞察

### 1.1 真实用户痛点矩阵


| 痛点分类     | 具体表现            | 解决方案               | 优先级 |
| -------- | --------------- | ------------------ | --- |
| **记忆流失** | 醒来5分钟内忘记70%梦境内容 | 极速录入(语音转文本) + 智能提示 | P0  |
| **表达困难** | "很乱,说不清楚"       | AI结构化引导 + 情绪光谱选择器  | P0  |
| **缺乏洞见** | 不知道梦在表达什么       | 多AI模型深度分析 + 符号解读   | P0  |
| **数据孤岛** | 记录后难以回顾和关联      | 智能标签 + 时空关联图谱      | P1  |
| **隐私焦虑** | 担心梦境内容泄露        | 私密Bucket + 分级隐私控制  | P1  |


### 1.2 产品核心价值主张

**不是日记本,而是潜意识的镜子**

- **捕捉层**: 5秒开始记录,支持语音/文字/图像多模态输入
- **理解层**: 基于现代神经科学与认知心理学的AI分析,识别情绪模式与睡眠质量关联
- **洞察层**: 潜意识情绪向量时间序列分析、重复元素识别、生理关联分析
- **成长层**: 个性化梦境报告、睡眠质量改善建议

## 二、数据模型设计(Backend Data Schema)

### 2.1 梦境核心表 (dreams)

**设计原则**:

1. **主表极度轻量** - 仅保留查询、列表展示必需字段
2. **统计字段提升** - 需要聚合的数据直接作为列存储
3. **大文本分离** - AI分析等大对象使用1:1关联表
4. **向量单独索引** - 便于 pgvector 高效检索

**质量评估算法** (后端计算):

```python
def calculate_sleep_score(quality: int, fragmented: bool, depth: int | None) -> float:
    """
    综合睡眠评分 (0-100)
    😴 5分 + 多次醒来 ≠ 真正高质量
    😐 3分 + 深睡 = 稳定型睡眠
    """
    base_score = quality * 20  # 20-100
    
    if fragmented:
        base_score -= 15  # 碎片化惩罚
    
    if depth:
        base_score += (depth - 2) * 10  # 深度奖励
    
    return max(0, min(100, base_score))
```

WITH trigger_dreams AS (
    SELECT d.id, d.dream_date
    FROM dreams d
    JOIN dream_triggers dt ON d.id = dt.dream_id
    JOIN triggers t ON dt.trigger_id = t.id
    WHERE t.trigger_key = 'spicy_food'
      AND d.user_id = :user_id
      AND d.dream_date >= CURRENT_DATE - INTERVAL '90 days'
),
nightmare_dreams AS (
    SELECT d.id
    FROM trigger_dreams td
    JOIN dreams d ON td.id = d.id
    JOIN dream_type_mappings dtm ON d.id = dtm.dream_id
    JOIN dream_types dt ON dtm.type_id = dt.id
    WHERE dt.type_name = 'NIGHTMARE'
)
SELECT 
    COUNT(DISTINCT td.id) as total_trigger_dreams,
    COUNT(DISTINCT nd.id) as nightmare_dreams,
    ROUND(COUNT(DISTINCT nd.id)::numeric / NULLIF(COUNT(DISTINCT td.id), 0) * 100, 2) as nightmare_rate
FROM trigger_dreams td
LEFT JOIN nightmare_dreams nd ON td.id = nd.id;

}

AWAKENING_STATES = ["NATURAL", "ALARM", "STARTLED", "GRADUAL"]

def calculate_sleep_score(
    quality: int,
    fragmented: bool,
    depth: int | None
) -> int:
    """
    综合睡眠评分 (0-100) - 优化版
    
    逻辑:
    - 基础分: quality * 20 (20-100)
    - 碎片化惩罚: * 0.85 (平滑衰减,而非硬性-15)
    - 深度奖励: (depth-2) * 8 (调整为8分,避免过度奖励)
    
    示例:
    - 😴 5 + 多次醒来 = 100 * 0.85 = 85 (不是满分)
    - 😐 3 + 深睡 = 60 + 8 = 68 (稳定型睡眠)
    - 😟 2 + 深睡 = 40 + 8 = 48 (质量差但深度好)
    """
    base_score = quality * 20
    
    # 碎片化平滑惩罚
    if fragmented:
        base_score *= 0.85
    
    # 深度奖励
    if depth:
        base_score += (depth - 2) * 8
    
    return int(max(0, min(100, base_score)))

class AIModelConfig:
    # OpenRouter 配置

# backend-v2/app/core/config.py
class Settings(BaseSettings):
    # ... 其他配置 ...
    
    # OpenRouter 配置 (从环境变量加载)
    openrouter_api_key: str = Field(..., alias="OPENROUTER_API_KEY")


# backend-v2/app/services/ai_service.py
class AIService:
    def __init__(self):
        # 统一使用 OpenRouter 客户端
        self.client = openai.AsyncOpenAI(
            base_url=AIModelConfig.OPENROUTER_BASE_URL,
            api_key=AIModelConfig.OPENROUTER_API_KEY,
        )
    
    async def transcribe_audio(self, audio_url: str) -> str:
        """语音转文字 - 使用 Gemini Flash (via OpenRouter)"""
        response = await self.client.chat.completions.create(
            model=AIModelConfig.MODELS["voice_to_text"],
            messages=[
                {
                    "role": "user",
                    "content": [
                        {"type": "audio_url", "audio_url": audio_url},
                        {"type": "text", "text": "请将音频转为文字"}
                    ]
                }
            ]
        )
        return response.choices[0].message.content
        
    async def analyze_image(self, image_url: str) -> dict:
        """图像分析 - 使用 Gemini Flash (via OpenRouter)"""
        response = await self.client.chat.completions.create(
            model=AIModelConfig.MODELS["image_analysis"],
            messages=[
                {
                    "role": "user",
                    "content": [
                        {"type": "image_url", "image_url": {"url": image_url}},
                        {"type": "text", "text": "描述这张图片的内容和情绪"}
                    ]
                }
            ]
        )
        return {"description": response.choices[0].message.content}
        
    async def analyze_dream_text(self, content: str) -> dict:
        """文本分析 - 使用 GPT-4o (via OpenRouter)"""
        response = await self.client.chat.completions.create(
            model=AIModelConfig.MODELS["text_analysis"],
            messages=[
                {"role": "system", "content": TEXT_ANALYSIS_PROMPT},
                {"role": "user", "content": content}
            ],
            temperature=0.7,
        )
        return response
        
    async def extract_symbols(self, content: str) -> list:
        """符号提取 - 使用 Gemini Flash (via OpenRouter)"""
        response = await self.client.chat.completions.create(
            model=AIModelConfig.MODELS["symbol_extraction"],
            messages=[
                {"role": "system", "content": SYMBOL_EXTRACTION_PROMPT},
                {"role": "user", "content": content}
            ],
            response_format={"type": "json_object"},
        )
        return json.loads(response.choices[0].message.content)
        
    async def generate_embedding(self, text: str) -> list[float]:
        """
        生成向量 - 使用 text-embedding-3-large (via OpenRouter)
        
        注意: OpenRouter 支持 OpenAI 的 embedding 接口
        """
        response = await self.client.embeddings.create(
            model=AIModelConfig.MODELS["embedding"],
            input=text,
            dimensions=AIModelConfig.EMBEDDING_DIMENSIONS,
        )
        return response.data[0].embedding
    
    async def generate_title(self, content: str) -> str:
        """生成梦境标题 - 使用 Gemini Flash (via OpenRouter)"""
        response = await self.client.chat.completions.create(
            model=AIModelConfig.MODELS["title_generation"],
            messages=[
                {"role": "system", "content": "根据梦境内容生成简短诗意的标题(10字以内)"},
                {"role": "user", "content": content}
            ],
            max_tokens=50,
        )
        return response.choices[0].message.content.strip()
```

# backend-v2/app/services/embedding_service.py
from openai import AsyncOpenAI

class EmbeddingService:
    def __init__(self):
        self.client = AsyncOpenAI(api_key=settings.openai_api_key)
    
    async def generate_embedding(self, text: str) -> list[float]:
        """
        生成文本的向量表示
        使用 text-embedding-3-small (1536维)
        """
        response = await self.client.embeddings.create(
            model="text-embedding-3-small",
            input=text,
        )
        return response.data[0].embedding
    
    async def semantic_search(
        self, 
        query: str, 
        user_id: str,
        limit: int = 10
    ) -> list[Dream]:
        """
        语义搜索梦境
        
        示例查询:
        - "我以前梦到过关于逃亡的主题吗？"
        - "找找我做过的飞行梦"
        """
        # 1. 生成查询向量
        query_embedding = await self.generate_embedding(query)
        
        # 2. 向量相似度搜索
        query = """
            SELECT *, 
                   1 - (content_embedding <=> :query_vector) AS similarity
            FROM dreams
            WHERE user_id = :user_id
              AND content_embedding IS NOT NULL
            ORDER BY content_embedding <=> :query_vector
            LIMIT :limit
        """
        
        results = await db.execute(
            query,
            {
                "query_vector": query_embedding,
                "user_id": user_id,
                "limit": limit
            }
        )
        
        return results.fetchall()
```
